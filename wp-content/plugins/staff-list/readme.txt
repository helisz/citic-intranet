== Description ==

**Staff List** - responsive WordPress plugin to showcase your team or staff members.

[View Demo](http://abcfolio.com/staff-list-live-preview-grid-image-left-text-right/)

Easilly create a wide range of directories and listings:

* Staff Directory
* List of Medical Personnel
* Member Directory
* Faculty and Staff Directory
* Medical Staff Directory
* Church Directory
* Directory of Doctors and Medical Staff
* Employee Directory
* Medical Personnel Directory
* Faculty Directory
* Realtors Directory
* Business Directory
* Staff List
* Department Directory
* Team Members List
* Organization Directory
* Lawyer Directory
* Attorney Directory
* Team Members Directory

Staff List is well suited to create both small and large staff directories.

[Live Previews](http://abcfolio.com/wordpress-plugin-staff-list-live-previews/)

== Features ==

**Staff List** is a highly configurable directory plugin to fit the unique requirements of your organization.

* Responsive layout.
* Single page option.
* Create data entry form in your own language.
* Custom templates. You can add any field in any order.
* Custom styles to create the uniform look and feel that match your theme.
* Easy to add to any page or post - just insert a shortcode.
* [Full list of features.](http://abcfolio.com/wordpress-plugin-staff-list-get-plugin/)
